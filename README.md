# Fill Box

<img width="1581" height="659" alt="Обложка" src="https://github.com/user-attachments/assets/2fd3b181-4250-4c1d-b259-fa00adc1fa2b" />

Fill Box - это сайт-редактор для создания пиксельной мозаики. В нём можно рисовать по сетке, менять размер поля, цвета активных и пустых ячеек, скругление, расстояние между плитками и эффект объёма.

Готовую мозаику можно скачать в форматах PNG или SVG.

## Что есть на сайте

- пресеты сетки: 4×4, 8×8, 9×9, 12×12, 16×16 и 32×32;
- режимы рисования и стирания;
- кнопки очистки, заполнения и инвертирования;
- настройка цветов, формы плиток и выпуклости;
- экспорт результата в PNG и SVG.
